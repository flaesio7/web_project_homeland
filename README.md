# Tripleten web_project_homeland

github pages
https://flaesio7.github.io/web_project_homeland/

Nome do projeto:(projeto Sprint 5: Design web adaptativo)

Descrição do projeto:
Este projeto consiste no desenvolvimento de uma página web responsiva para o projeto de Pátria para Pátria para apresentar diferentes visuais a ser visto,realizado durante a sprint 5 do bootcamp da tripleten.

Tecnologias usadas 
html5,css3,Flexbox,Media queries.

Funcionalidades:
Projeto visado para tamanho máximo em 1280px (desktop),768px(Tablet) tamanho mínimo em 320px(mobile) designado para alguns tamanhos de telas específicos seguindo a metodologia BEM.
